# DiPAMIA

Aplicación Web DiPAMIA

## Guía de Uso

**Servidor**
- Ir a la carpeta de servidor: ```cd /server```
- Instalar dependencias: ```pnpm install```
- Ejecutar el servidor: ```pnpm run dev```

**Cliente**
- Ir a la carpeta de cliente: ```cd /../client```
- Instalar dependencias: ```pnpm install```
- Ejecutar el cliente: ```pnpm run dev```
